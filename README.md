- Responsive design
- Contact form
- Newsletter subscription
- Team information
- Services overview

## Pages

- Home (index.html)
- About (about.html)
- Services (services.html)
- Contact (contact.html)

## Technologies Used

- HTML5
- CSS3
- JavaScript
